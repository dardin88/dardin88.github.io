aDoctor is a tool able to identify 15 Android-specific code smells from the catalogue by Reimann et al.
It is built on top of the Eclipse Java Development Toolkit (JDT)
aDoctor is released under MIT license and it is hosted here.

PETrA is a software able to estimate the energy consumption of method calls in Android apps.
It is based on some Android tools that are Monkey, Batterystats, Systrace, and dmtracedump.
PETrA is available here.

Landfill is a Web-based platform for sharing code smell datasets. It also provides a set of APIs for programmatically accessing its data.
Anyone can contribute by: improving existing datasets or sharing and posting new datasets.
Landfill is available at http://www.sesa.unisa.it/landfill/.

Gnome Maps is a map application for GNOME. More information on https://wiki.gnome.org/Apps/Maps

GraphHopper offers memory efficient algorithms in Java for routing on graphs. E.g. Dijkstra and A* but also optimized road routing algorithms like Contraction Hierarchies. It stands under the Apache License and is build on a large test suite.
More information on https://graphhopper.com
